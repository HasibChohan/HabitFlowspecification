# ⚡ HabitFlow — Smart Habit Tracking Dashboard

> A modern, AI-assisted habit tracking web app built with React + Vite + Tailwind CSS

**Developed by: Muhammad Haseeb** | Built using Vibe Coding (AI-Assisted Development)

## 🚀 Live Demo

**[habitflow.vercel.app](https://habitflow.vercel.app)** ← Replace with your Vercel URL after deployment

---

## ✨ Features

- **Habit Manager** — Add, edit, delete habits with custom emoji icons
- **Streak Tracking** — Consecutive day streaks with all-time best records
- **Analytics Dashboard** — 7-day bar charts + completion rates per habit
- **Calendar View** — Monthly grid showing completion history
- **Achievement System** — Unlock badges as you build consistency
- **Daily Quotes** — Rotating motivational quotes
- **Glassmorphism UI** — Dark, modern design with smooth animations
- **Fully Responsive** — Works perfectly on mobile, tablet, and desktop
- **Local Storage** — Data persists between sessions, no login needed

## 📦 Installation & Setup

```bash
# 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/habitflow.git
cd habitflow

# 2. Install dependencies
npm install

# 3. Start development server
npm run dev

# 4. Open in browser
# http://localhost:5173
```

---

## 🌐 Deployment (Vercel)

### Method 1 — Vercel Dashboard (Recommended)
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project
3. Import your `habitflow` GitHub repository
4. Click **Deploy** — done! ✅

### Method 2 — Vercel CLI
```bash
npm install -g vercel
vercel
```
